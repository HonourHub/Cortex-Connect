# CortexConnect

CortexConnect is an interactive neuroscience education platform built with HTML, CSS, and JavaScript to make complex brain science more accessible through visual explanations, simulations, and quizzes.

## Live Demo
[Visit CortexConnect](YOUR_GITHUB_PAGES_LINK)

## Project Vision
I created CortexConnect as a passion project at the intersection of neuroscience, education, and technology. My goal is to make topics such as action potentials, brain structure, and neuroplasticity easier to understand for students through engaging design and interaction.

## Why I Built This
I’m deeply interested in neuroscience, medicine, engineering, and science communication. CortexConnect allows me to combine those interests while also strengthening my front-end development, UI/UX, and digital storytelling skills.

## Features
- Responsive multi-page website
- Interactive neuroscience modules
- Action potential animation
- Clickable brain structure exploration
- Neuroplasticity learning simulation
- Quizzes and reflection prompts
- Accessible semantic HTML structure
- Live deployment with GitHub Pages

## Modules
### 1. Action Potentials
Explains how neurons fire using a visual phase-based interaction and quiz.

### 2. Brain Structure
Introduces major brain regions with an interactive diagram and info panels.

### 3. Neuroplasticity
Shows how repeated activity strengthens neural connections through an interactive simulation.
